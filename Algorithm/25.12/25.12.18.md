## 力扣：3652. 按策略买卖股票的最佳时机
**链接**：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-using-strategy/  
**相关标签**：`数组` `前缀和` `滑动窗口`  
**难度**：`中等`
### 方法1：`前缀和`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    long long maxProfit(vector<int>& prices, vector<int>& strategy, int k) {
        int n = prices.size();
        vector<long long> pricesSum(n+1), profits(n+1);
        pricesSum[0] = 0, profits[0] = 0;
        for(int i=0; i<n; i++){
            pricesSum[i+1] = prices[i] + pricesSum[i];
            profits[i+1] = 1LL * prices[i] * strategy[i] + profits[i];
        }
        long long ans = profits[n];
        for(int i=1; i<=n-k+1; i++)
            ans = max(ans, profits[i-1] + pricesSum[i+k-1] - pricesSum[i+k/2-1] + profits[n] - profits[i+k-1]);
        return ans;
    }
};
```
