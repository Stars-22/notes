## 力扣：51. N 皇后
**链接**：https://leetcode.cn/problems/n-queens/  
**相关标签**：`数组` `回溯`  
**难度**：`困难`
### 方法1：`深度优先搜索`
- **时间复杂度**：$O(n!)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    vector<vector<string>> solveNQueens(int n) {
        vector<bool> y(n, false), p(2*n-1, false), q(2*n-1, false);
        vector<vector<string>> ans;
        vector<string> temp(n, string(n, '.'));
        auto dfs = [&](this auto&& dfs, int x) -> void {
            if(x == n){
                ans.push_back(temp);
                return;
            }
            for(int i=0; i<n; i++){
                if(y[i] || p[x+i] || q[i-x+n-1]) continue;
                p[x+i] = q[i-x+n-1] = y[i] = true;
                temp[x][i] = 'Q';
                dfs(x+1);
                temp[x][i] = '.';
                p[x+i] = q[i-x+n-1] = y[i] = false;
            }
        };
        dfs(0);
        return ans;
    }
};
```

## 洛谷：P14359 [CSP-J 2025] 异或和
**链接**：https://www.luogu.com.cn/problem/P14359/  
**相关标签**：`动态规划 DP` `贪心` `哈希 hashing` `前缀和` `位运算` `CSP-J 入门级` `2025` `O2优化`  
**难度**：`普及/提高−`
### 方法1：`贪心+前缀和+哈希`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n, k, a, ans = 0, sum = 0;
	cin >> n >> k; 
	unordered_set<int> hset;
	for(int i=0; i<n; i++){
		cin >> a;
		hset.insert(sum);
		sum ^= a;
		if(hset.find(sum^k) != hset.end()){
			ans++;
			hset.clear();
		}
	}
	cout << ans;
	return 0;
}
```
