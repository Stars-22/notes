## 力扣：944. 删列造序
**链接**：https://leetcode.cn/problems/delete-columns-to-make-sorted/  
**相关标签**：`数组` `字符串`  
**难度**：`简单`
### 方法1：`模拟`
- **时间复杂度**：$O(nm)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int minDeletionSize(vector<string>& strs) {
        int ans = 0;
        for(int i=0; i<strs[0].size(); i++){
            for(int j=1; j<strs.size(); j++){
                if(strs[j][i] < strs[j-1][i]){
                    ans++;
                    break;
                }
            }
        }
        return ans;
    }
};
```
