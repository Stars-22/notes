## 力扣：152. 乘积最大子数组
**链接**：https://leetcode.cn/problems/maximum-product-subarray/  
**相关标签**：`数组` `动态规划`  
**难度**：`中等`
### 方法1：`动态规划`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        long lmax = nums[0], lmin = nums[0], ans = nums[0];
        for(int i=1; i<nums.size(); i++){
            long lmaxTemp = max(lmax*nums[i], max((long)nums[i], lmin*nums[i]));
            lmin = min(lmin*nums[i], min((long)nums[i], lmax*nums[i]));
            lmax = lmaxTemp;
            ans = max(ans, lmax);
        }
        return ans;
    }
};
```

## 力扣：416. 分割等和子集
**链接**：https://leetcode.cn/problems/partition-equal-subset-sum/  
**相关标签**：`数组` `动态规划`  
**难度**：`中等`
### 方法1：`动态规划`
- **时间复杂度**：$O(n*sum)$
- **空间复杂度**：$O(sum)$
```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = accumulate(nums.begin(), nums.end(), 0);
        if(sum & 1) return false;
        sum /= 2;
        vector<bool> dp(sum+1, false);
        dp[0] = true;
        for(const auto& n : nums)
            for(int i=sum; i>=n; i--)
                dp[i] = dp[i] || dp[i-n];
        return dp[sum];
    }
};
```
### 方法2：`动态规划+位运算`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(sum)$
```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = accumulate(nums.begin(), nums.end(), 0);
        if(sum & 1) return false;
        sum /= 2;
        bitset<20001> dp = {1};
        for(int x: nums) dp |= dp << x;
        return dp[sum];
    }
};
```
