## 力扣：2574. 左右元素和的差值
**链接**：https://leetcode.cn/problems/left-and-right-sum-differences/  
**相关标签**：`数组` `前缀和`  
**难度**：`简单`
### 方法1：`前缀和`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    vector<int> leftRightDifference(vector<int>& nums) {
        vector<int> ans(nums.size(), 0);
        for(int i=0; i<nums.size()-1; i++) ans[i+1] = ans[i] + nums[i];
        int sum = 0;
        for(int i=nums.size()-2; i>=0; i--){
            sum += nums[i+1];
            ans[i] = abs(ans[i] - sum);
        }
        return ans;
    }
};
```
