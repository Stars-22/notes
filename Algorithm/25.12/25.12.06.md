## 力扣：62. 不同路径
**链接**：https://leetcode.cn/problems/unique-paths/  
**相关标签**：`数学` `动态规划` `组合数学`  
**难度**：`中等`
### 方法1：`动态规划`
- **时间复杂度**：$O(mn)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<int> dp(n+1, 1);
        dp[0] = 0;
        for(int i=1; i<m; i++){
            for(int j=1; j<=n; j++){
                dp[j] = dp[j-1] + dp[j];
            }
        }
        return dp[n];
    }
};
```

## 力扣：64. 最小路径和
**链接**：https://leetcode.cn/problems/minimum-path-sum/  
**相关标签**：`数组` `动态规划` `矩阵`  
**难度**：`中等`
### 方法1：`动态规划`
- **时间复杂度**：$O(mn)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        int m = grid.size(), n = grid[0].size();
        vector<int> dp(n+1, INT_MAX);
        dp[1] = 0;
        for(int i=0; i<m; i++){
            for(int j=1; j<=n; j++){
                dp[j] = min(dp[j-1], dp[j]) + grid[i][j-1];
            }
        }
        return dp[n];
    }
}p;
```
