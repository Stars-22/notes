## 力扣：295. 数据流的中位数
**链接**：https://leetcode.cn/problems/find-median-from-data-stream/  
**相关标签**：`设计` `双指针` `数据流` `排序` `堆（优先队列）`  
**难度**：`困难`
### 方法1：`优先队列`
- **时间复杂度**：
  - 初始化: $O(1)$
  - addNum: $O(\log n)$
  - findMedian: $O(1)$
- **空间复杂度**：$O(n)$
```cpp
class MedianFinder {
    priority_queue<int> l;
    priority_queue<int, vector<int>, greater<int>> r;
public:
    MedianFinder() {}
    void addNum(int num) {
        l.push(num);
        r.push(l.top());
        l.pop();
        if(l.size() < r.size()){
            l.push(r.top());
            r.pop();
        }
    }
    double findMedian() {
        if(l.size() == r.size() + 1) return l.top();
        return (l.top() + r.top()) / 2.0;
    }
};
```
