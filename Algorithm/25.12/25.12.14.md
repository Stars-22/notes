## 力扣：2147. 分隔长廊的方案数
**链接**：https://leetcode.cn/problems/number-of-ways-to-divide-a-long-corridor/  
**相关标签**：`数学` `字符串` `动态规划`  
**难度**：`困难`
### 方法1：`乘法原理`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int numberOfWays(string corridor) {
        int ans = 1, sum = 0, num = 0, last = 0;
        for(char c : corridor) if(c == 'S') num++;
        if(num == 0 || num % 2 == 1) return 0;
        for(int i=0; i<corridor.size(); i++){
            if(corridor[i] == 'S'){
                sum++;
                if(sum%2 == 0){
                    if(sum == num) break;
                    int temp = 1;
                    for(; corridor[i+1]=='P'; i++) temp++;
                    ans = (1LL * ans * temp) % 1000000007;
                }
            }
        }
        return ans;
    }
};
```
