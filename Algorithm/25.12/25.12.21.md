## 力扣：955. 删列造序 II
**链接**：https://leetcode.cn/problems/delete-columns-to-make-sorted-ii/  
**相关标签**：`贪心` `数组` `字符串`  
**难度**：`中等`
### 方法1：`贪心`
- **时间复杂度**：$O(nm)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    int minDeletionSize(vector<string>& strs) {
        int ans = 0, n = strs.size(), m = strs[0].size();
        vector<bool> flag(n, false);
        for(int i=0; i<m; i++){
            int temp = ans;
            for(int j=1; j<n; j++){
                if(!flag[j] && strs[j][i] < strs[j-1][i]){
                    ans++;
                    break;
                }
            }
            if(temp == ans)
                for(int j=1; j<n; j++)
                    if(strs[j][i] != strs[j-1][i]) flag[j] = true;
        }
        return ans;
    }
};
```
