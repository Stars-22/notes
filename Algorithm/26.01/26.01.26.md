# 26.01.26 算法刷题笔记

## 洛谷：P1303 A*B Problem
**链接**：https://www.luogu.com.cn/problem/P1303/  
**相关标签**：`模拟` `字符串` `高精度`  
**难度**：`普及-`
### 方法1：`模拟+预处理`
- **时间复杂度**：$O(mn)$
- **空间复杂度**：$O(max(m,n))$
```cpp
#include "bits/stdc++.h"
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    char a[2002], b[2002], x[10][2002], a_[2002], b_[2002];
    int k[10];
    cin >> a >> b;
    for(int i=strlen(a)-1; i>=0; i--) a_[strlen(a)-i-1] = a[i] - '0';
    for(int i=strlen(b)-1; i>=0; i--) b_[strlen(b)-i-1] = b[i] - '0';
    for(int i=1; i<10; i++) {
        int temp = 0;
        for(int j=0; j<strlen(a); j++){
            x[i][j] = a_[j] * i + temp;
            temp = (x[i][j]) / 10;
            x[i][j] %= 10;
        }
        if(temp){
            x[i][strlen(a)] = temp;
            k[i] = strlen(a) + 1;
        }
        else k[i] = strlen(a);
    }
    int ans[4004], ans_len = 0;
    memset(ans, 0, sizeof(ans));
    for(int i=0; i<strlen(b); i++) {
        if(b_[i] == 0) continue;
        for(int j=0; j<k[b_[i]]; j++) {
            ans[i+j] +=x[b_[i]][j];
            ans[i+j+1] += ans[i+j] / 10;
            ans[i+j] %= 10;
        }
    }
    for(int i=4003; i>=0; i--) {
        if(ans[i] != 0 || i == 0) {
            for(int j=i; j>=0; j--) cout << (char)(ans[j] + '0');
            return 0;
        }
    }
    return 0;
}
```
