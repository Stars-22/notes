## 力扣：230. 二叉搜索树中第 K 小的元素
**链接**：https://leetcode.cn/problems/kth-smallest-element-in-a-bst/  
**相关标签**：`树` `深度优先搜索` `二叉搜索树` `二叉树`  
**难度**：`中等`
### 方法1：`中序遍历`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(h)$
```cpp
class Solution {
public:
    int cur = 0;
    int kthSmallest(TreeNode* root, int k) {
        if(root->left){
            int ans = kthSmallest(root->left, k);
            if(ans != -1) return ans;
        }
        if(++cur == k) return root->val;
        if(root->right){
            int ans = kthSmallest(root->right, k);
            if(ans != -1) return ans;
        }
        return -1;
    }
};
```

## 力扣：763. 划分字母区间
**链接**：https://leetcode.cn/problems/partition-labels/  
**相关标签**：`贪心` `哈希表` `双指针` `字符串`  
**难度**：`中等`
### 方法1：`贪心`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(|Σ|)$ 其中Σ是字符集，本题中字符串只包含英文字母，|Σ|=26
```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        vector<int> last(26, 0), ans;
        int start = -1, end = 0;
        for(int i=0; i<s.size(); i++) last[s[i] - 'a'] = i;
        for(int i=0; i<s.size(); i++){
            end = max(end, last[s[i] - 'a']);
            if(end == i){
                ans.push_back(end - start);
                start = i;
            }
        }
        return ans;
    }
};
```
