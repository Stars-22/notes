## 力扣：208. 实现 Trie (前缀树)
**链接**：https://leetcode.cn/problems/implement-trie-prefix-tree/  
**相关标签**：`设计` `字典树` `哈希表` `字符串`  
**难度**：`中等`
### 方法1：`字典树`
- **时间复杂度**：
  - 初始化: $O(1)$
  - insert: $O(L)$
  - search: $O(L)$
  - startsWith: $O(L)$
- **空间复杂度**：$O(|T|*Σ)$
```cpp
class Trie {
private:
    struct Tree{
        vector<Tree*> son;
        bool isEnd;
        Tree() : son(26, nullptr), isEnd(false) {}
        ~Tree(){ for(const auto& s : son) if(s) delete s; }
    } root;
    Tree* searchPrefix(const string& word){
        Tree* cur = &root;
        for(int k = 0; k<word.size(); k++){
            if(cur->son[word[k] - 'a'] == nullptr) return nullptr;
            cur = cur->son[word[k] - 'a'];
        }
        return cur;
    }
public:
    Trie() { }
    void insert(string word) {
        Tree* cur = &root;
        for(int k = 0; k<word.size(); k++){
            if(cur->son[word[k] - 'a'] == nullptr) cur->son[word[k] - 'a'] = new Tree();
            cur = cur->son[word[k] - 'a'];
        }
        cur->isEnd = true;
    }
    bool search(string word) {
        Tree* cur = searchPrefix(word);
        return cur != nullptr && cur->isEnd;
    }
    bool startsWith(string prefix) {
        return searchPrefix(prefix) != nullptr;
    }
};
```
