# 25.11.08 算法刷题笔记

## 力扣：42. 接雨水
**链接**：https://leetcode.cn/problems/trapping-rain-water/  
**相关标签**：`栈` `数组` `双指针` `动态规划` `单调栈`  
**难度**：`困难`
### 方法1：`双指针`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int l = 0, r = height.size()-1, ans = 0, t = 0;
        while(true){
            while(l < r && (height[l] <= height[l+1] || height[l] <= t))
                if(++l<r) ans -= min(t, height[l]);
            while(l<r && (height[r] <= height[r-1] || height[r] <= t))
                if(l<--r) ans -= min(t, height[r]);
            if(l >= r) break;
            ans += (min(height[l], height[r]) - t) * (r - l - 1);
            t = min(height[l], height[r]);
        }
        return ans;
    }
};
```
