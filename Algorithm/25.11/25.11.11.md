## 力扣：48. 旋转图像
**链接**：https://leetcode.cn/problems/rotate-image/  
**相关标签**：`数组` `数学` `矩阵`  
**难度**：`中等`
### 方法1：`原地算法`
- **时间复杂度**：$O(n^2)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        for(int i=0; i<matrix.size()/2; i++){
            for(int j=i; j<matrix.size()-i-1; j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[matrix.size()-j-1][i];
                matrix[matrix.size()-j-1][i] = matrix[matrix.size()-i-1][matrix.size()-j-1];
                matrix[matrix.size()-i-1][matrix.size()-j-1] = matrix[j][matrix.size()-i-1];
                matrix[j][matrix.size()-i-1] = temp;
            }
        }
    }
};
```

## 力扣：238. 除自身以外数组的乘积
**链接**：https://leetcode.cn/problems/product-of-array-except-self/  
**相关标签**：`数组` `前缀和`  
**难度**：`中等`
### 方法1：`左右乘积法`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int> ans(nums.size());
        ans[0] = 1;
        for(int i=1; i<nums.size(); i++) ans[i] = nums[i-1] * ans[i-1];
        int m = 1;
        for(int i=nums.size()-1; i>=0; i--){
            ans[i] *= m;
            m *= nums[i];
        }
        return ans;
    }
};
```
