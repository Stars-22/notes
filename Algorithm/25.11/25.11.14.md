## 力扣：84. 柱状图中最大的矩形
**链接**：https://leetcode.cn/problems/largest-rectangle-in-histogram/  
**相关标签**：`栈` `数组` `单调栈`  
**难度**：`困难`
### 方法1：`单调栈`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        stack<int> c;
        int ans = 0;
        heights.insert(heights.begin(), 0);
        heights.push_back(0);
        c.push(heights[0]);
        for(int r=1; r<heights.size(); r++){
            while(heights[r] < heights[c.top()]){
                int cur = c.top();
                c.pop();
                ans = max(ans, heights[cur] * (r - c.top() - 1));
            }
            c.push(r);
        }
        return ans;
    }
};
```
