## 力扣：23. 合并 K 个升序链表
**链接**：https://leetcode.cn/problems/merge-k-sorted-lists/  
**相关标签**：`链表` `分治` `堆（优先队列）` `归并排序`  
**难度**：`困难`
### 方法1：`分治（迭代）`
- **时间复杂度**：$O(n \log k)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    ListNode* merge2Lists(ListNode* a, ListNode* b) {
        ListNode ans;
        ListNode* cur = &ans;
        while(a && b){
            if(a->val < b->val){
                cur->next = a;
                a = a->next;
            }
            else{
                cur->next = b;
                b = b->next;
            }
            cur = cur->next;
        }
        cur->next = a ? a : b;
        return ans.next;
    }
    ListNode* mergeKLists(vector<ListNode*>& lists) {
        if (lists.empty()) return nullptr;
        for (int i = 1; i < lists.size(); i *= 2) {
            for (int j = 0; j + i < lists.size(); j += 2 * i) {
                lists[j] = merge2Lists(lists[j], lists[j + i]);
            }
        }
        return lists[0];
    }
};
```
