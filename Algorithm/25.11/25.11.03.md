## 力扣：2. 两数相加
**链接**：https://leetcode.cn/problems/add-two-numbers/  
**相关标签**：`递归` `链表` `数学`  
**难度**：`中等`
### 方法1：`模拟`
- **时间复杂度**：$O(max(m,n))$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode ans;
        ListNode* cur = &ans;
        int k = 0;
        while(l1 || l2){
            cur->next = new ListNode();
            cur = cur->next;
            cur->val = (l1?l1->val:0) + (l2?l2->val:0) + k;
            k = 0;
            if(cur->val >= 10){
                cur->val -= 10;
                k = 1;
            }
            l1 = l1?l1->next:nullptr;
            l2 = l2?l2->next:nullptr;
        }
        if(k) cur->next = new ListNode(k);
        return ans.next;;
    }
};
```

## 力扣：101. 对称二叉树
**链接**：https://leetcode.cn/problems/symmetric-tree/  
**相关标签**：`树` `深度优先搜索` `广度优先搜索` `二叉树`  
**难度**：`简单`
### 方法1：`迭代`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        vector<TreeNode*> stack;
        stack.push_back(root->left);
        stack.push_back(root->right);
        while(!stack.empty()){
            TreeNode* cura = stack.back(); stack.pop_back();
            TreeNode* curb = stack.back(); stack.pop_back();
            if (!cura && !curb) continue;
            if (!cura || !curb || cura->val != curb->val) return false;
            stack.push_back(cura->left);
            stack.push_back(curb->right);
            stack.push_back(cura->right);
            stack.push_back(curb->left);
        }
        return true;
    }
};
```
