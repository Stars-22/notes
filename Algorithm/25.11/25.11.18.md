## 力扣：45. 跳跃游戏 II
**链接**：https://leetcode.cn/problems/jump-game-ii/  
**相关标签**：`贪心` `数组` `动态规划`  
**难度**：`中等`
### 方法1：`贪心`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        int f = 0, ans = 0, cur = 0;
        while(f < nums.size() - 1){
            int f_cur = f;
            for(int i=cur; i<=f_cur; i++){
                if(f < i+nums[i]){
                    cur = i;
                    f = i+nums[i];
                }
            }
            ans++;
        }
        return ans;
    }
};
```

## 力扣：287. 寻找重复数
**链接**：https://leetcode.cn/problems/find-the-duplicate-number/  
**相关标签**：`位运算` `数组` `双指针` `二分查找`  
**难度**：`中等`
### 方法1：`快慢指针`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int findDuplicate(vector<int>& nums) {
        int fast = 0, slow = 0;
        do{
            fast = nums[nums[fast]];
            slow = nums[slow];
        } while(fast != slow);
        slow = 0;
        while(fast != slow){
            fast = nums[fast];
            slow = nums[slow];
        }
        return fast;
    }
};
```
