## 力扣：31. 下一个排列
**链接**：https://leetcode.cn/problems/next-permutation/  
**相关标签**：`数组` `双指针`  
**难度**：`中等`
### 方法1：`字典序交换反转`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int s = 0, e = nums.size()-1;
        for(int i=e-1; i>=0; i--){
            if(nums[i] >= nums[i+1]) continue;
            for(int j=e; j>=0; j--){
                if(nums[j] > nums[i]){
                    swap(nums[i], nums[j]);
                    break;
                }
            }
            s = i+1;
            break;
        }
        for(int i=0; i<(e-s+1)/2; i++) swap(nums[e-i], nums[s+i]);
    }
};
```

## 力扣：207. 课程表
**链接**：https://leetcode.cn/problems/course-schedule/  
**相关标签**：`深度优先搜索` `广度优先搜索` `图` `拓扑排序`  
**难度**：`中等`
### 方法1：`深度优先搜索`
- **时间复杂度**：$O(n+m)$
- **空间复杂度**：$O(n+m)$
```cpp
class Solution {
public:
    vector<vector<int>> a;
    vector<int> vis;
    bool dfs(int i){
        if(vis[i] == 1) return false;
        if(vis[i] == 2) return true;
        vis[i] = 1;
        for(const auto& o : a[i])
            if(!dfs(o)) return false;
        vis[i] = 2;
        return true;
    }
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        a.resize(numCourses);
        vis.resize(numCourses);
        for(const auto& p : prerequisites) a[p[1]].push_back(p[0]);
        for(int i=0; i<numCourses; i++) if(!dfs(i)) return false;
        return true;
    }
};
```
