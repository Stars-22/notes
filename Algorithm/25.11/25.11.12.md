## 力扣：41. 缺失的第一个正数
**链接**：https://leetcode.cn/problems/first-missing-positive/  
**相关标签**：`数组` `哈希表`  
**难度**：`困难`
### 方法1：`原地哈希`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int firstMissingPositive(vector<int>& nums) {
        for(auto& n : nums)
            if(n <= 0 || n > nums.size()) n = nums.size() + 1;
        for(const auto& n : nums)
            if(abs(n) <= nums.size()) nums[abs(n)-1] = -abs(nums[abs(n)-1]);
        for(int i=0; i<nums.size(); i++)
            if(nums[i] > 0) return i+1;
        return nums.size()+1;
    }
};
```
