# 25.11.10 算法刷题笔记

## 力扣：76. 最小覆盖子串
**链接**：https://leetcode.cn/problems/minimum-window-substring/  
**相关标签**：`哈希表` `字符串` `滑动窗口`  
**难度**：`困难`
### 方法1：`双指针`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    string minWindow(string s, string t) {
        int start = 0, minLen = INT_MAX, required = t.size();
        deque<int> q;
        unordered_map<char, int> a;
        for(const auto& c : t) a[c]++;
        for(int i=0; i<s.size(); i++){
            if(a.find(s[i]) != a.end()){
                if(a[s[i]]-- > 0) required--;
                q.push_back(i);
            }
            while(required == 0){
                if(q.back() - q.front() < minLen){
                    start = q.front();
                    minLen = q.back() - q.front() + 1;
                }
                if(a[s[q.front()]]++ >= 0) required++;
                q.pop_front();
            }
        }
        return minLen == INT_MAX ? "" : s.substr(start, minLen);
    }
};
```
