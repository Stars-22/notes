## 力扣：22. 括号生成
**链接**：https://leetcode.cn/problems/generate-parentheses/  
**相关标签**：`字符串` `动态规划` `回溯`  
**难度**：`中等`
### 方法1：`深度优先搜索`
- **时间复杂度**：$O(4^n / \sqrt n)$
- **空间复杂度**：$O(n)$
```cpp
class Solution {
public:
    int a[8], n;
    vector<string> ans;
    void creatAns(){
        string s;
        for(int i=0; i<n; i++) s += '(' + string(a[i], ')');
        ans.push_back(move(s));
    }
    void dfs(int k, int n){
       for(int i=k<n?1:0; i<=n; i++){
           a[k] = i;
           if(i == n) creatAns();
           else dfs(k-1, n-i);
       }
       a[k] = 0;
    }
    vector<string> generateParenthesis(int n) {
        this->n = n;
        dfs(n-1, n);
        return ans;
    }
};
```

## 力扣：461. 汉明距离
**链接**：https://leetcode.cn/problems/hamming-distance/  
**相关标签**：`位运算`  
**难度**：`简单`
### 方法1：`移位实现位计数`
- **时间复杂度**：$O(\log C)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    int hammingDistance(int x, int y) {
        int k = x ^ y, ans = 0;
        do{ if(k&1) ans++; } while(k >>= 1);
        return ans;
    }
};
```
