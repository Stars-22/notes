# 25.11.27 算法刷题笔记

## 力扣：25. K 个一组翻转链表
**链接**：https://leetcode.cn/problems/reverse-nodes-in-k-group/  
**相关标签**：`递归` `链表`  
**难度**：`困难`
### 方法1：`原地算法`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        int n = 0;
        ListNode ans;
        ans.next = head;
        ListNode* tail = &ans;
        while(head){
            n++;
            if(n % k == 0){
                ListNode* next = head->next;
                ListNode* h = tail->next;
                ListNode* t = tail->next;
                ListNode* last = next;
                ListNode* cur = tail->next;
                while(cur != next){
                    ListNode* next = cur->next;
                    h = cur;
                    cur->next = last;
                    last = cur;
                    cur = next;
                }
                tail->next =  h;
                tail = t;
                head = tail;
            }
            head = head->next;
        }
        return ans.next;
    }
};
```
