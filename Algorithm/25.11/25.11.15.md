# 25.11.15 算法刷题笔记

## 力扣：75. 颜色分类
**链接**：https://leetcode.cn/problems/sort-colors/  
**相关标签**：`数组` `双指针` `排序`  
**难度**：`中等`
### 方法1：`双指针`
- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(1)$
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int l = 0, r = nums.size() - 1;
        for(int i=l; i<=r; i++){
            if(nums[i] == 0 && i != l) swap(nums[l++], nums[i--]);
            else if(nums[i] == 2 && i != r) swap(nums[r--], nums[i--]);
        }
    }
};
```
